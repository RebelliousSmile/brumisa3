Class: GenericDocument

GenericDocument(theme)

Document générique - Structure universelle (titres, paragraphes, listes) Fonctionne avec tous les systèmes de jeu via SystemTheme

Usage :

  • Documents de règles
  • Guides et manuels
  • Suppléments narratifs
  • Tout contenu avec structure classique
