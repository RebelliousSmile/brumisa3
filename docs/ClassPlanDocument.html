ClassPlanDocument(theme)

Document Plan de Classe - Layout spécifique pour les classes de personnages Structure prédéfinie avec sections caractéristiques des plans de classe

Usage :

  • Plans de classe Monsterhearts (Chosen, Ghost, Mortal, etc.)
  • Classes d'autres systèmes avec structure similaire
  • Documents avec sections fixes et mise en page spécialisée
