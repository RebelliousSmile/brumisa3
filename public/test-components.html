Hero Section avec Thèmes Système

Exemples d'utilisation du hero-section avec différents systèmes de jeu. En production, ce composant serait utilisé avec le moteur de templates EJS.

Exemples d'usage :

Monsterhearts - Gothic Romance

<%- include('partials/hero-section', {
    heroTitle: 'Monsterhearts',
    heroSubtitle: 'Un jeu sur les monstres, leurs secrets et leur scolarité',
    heroBadge: { text: 'Gothic Romance', variant: 'primary' },
    heroCTA: [
        { text: 'Créer un personnage', variant: 'primary', size: 'lg' },
        { text: 'Voir les règles', variant: 'outline', size: 'lg' }
    ],
    theme: 'gradient',
    gameSystem: 'monsterhearts'
}) %>

Engrenages - Steampunk

<%- include('partials/hero-section', {
    heroTitle: 'Engrenages',
    heroSubtitle: 'L\'aventure steampunk dans un monde de vapeur et d\'acier',
    heroBadge: { text: 'Steampunk', variant: 'primary' },
    heroCTA: [
        { text: 'Commencer l\'aventure', variant: 'primary', size: 'lg' },
        { text: 'Explorer le monde', variant: 'outline', size: 'lg' }
    ],
    theme: 'gradient',
    gameSystem: 'engrenages'
}) %>

Metro 2033 - Post-Apocalyptic

<%- include('partials/hero-section', {
    heroTitle: 'Metro 2033',
    heroSubtitle: 'Survivez dans les tunnels d\'un monde post-apocalyptique',
    heroBadge: { text: 'Post-Apocalyptic', variant: 'primary' },
    heroCTA: [
        { text: 'Entrer dans le métro', variant: 'primary', size: 'lg' },
        { text: 'Guide de survie', variant: 'outline', size: 'lg' }
    ],
    theme: 'gradient',
    gameSystem: 'metro2033'
}) %>
